# TSPVisualizer
- Visualize Traveling Salesman Problem(TSP)
- Using C++ with OpenGL

## File Format
- Use file in [TSPLIB](https://www.iwr.uni-heidelberg.de/groups/comopt/software/TSPLIB95/)

## Compile Command
in OS X
```bash
g++ main.cpp showTSP.cpp -framework GLUT -framework OpenGL
```

## Run
```bash
./a.out a280.tsp
```
